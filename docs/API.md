# PlanIQ — REST API Contracts

Base URL: `/api/v1`. JSON everywhere. Auth via `Authorization: Bearer <access>` (or httpOnly refresh cookie for `/auth/refresh`). All list endpoints support `?page&limit&sort&q`. Errors use a uniform envelope.

### Error envelope
```json
{ "error": { "code": "VALIDATION_ERROR", "message": "…", "details": [{ "field": "email", "issue": "invalid" }], "traceId": "…" } }
```
Status codes: `400` validation, `401` unauthenticated, `403` forbidden, `404` not found, `409` conflict, `413` payload too large, `422` AI/processing, `429` rate limited, `500` server.

Interactive docs: **Swagger UI at `/api/docs`** (OpenAPI 3 generated by NestJS). Postman collection in `docs/planiq.postman.json`.

## Auth  `/auth`
| Method | Path | Body | Returns |
|--------|------|------|---------|
| POST | `/register` | `{tenantName,name,email,password}` | `{user, accessToken}` + refresh cookie |
| POST | `/login` | `{email,password}` | `{user, accessToken}` + refresh cookie |
| POST | `/refresh` | – (cookie) | `{accessToken}` (rotates refresh) |
| POST | `/logout` | – | `204` (revokes session) |
| GET | `/me` | – | `{user, tenant, permissions}` |
| POST | `/password/forgot` | `{email}` | `202` |
| POST | `/password/reset` | `{token,password}` | `204` |

### Upload (presigned, direct-to-S3)
| POST | `/projects/:id/floors/upload-url` | `{fileName,mime,sizeBytes,floorId?}` → `{assetId, uploadUrl, s3Key, headers}` |
| POST | `/assets/:assetId/complete` | – → verifies object, for PDFs auto-creates floors per page, enqueues analysis → `{floors[], jobIds[]}` |
| GET | `/assets/:assetId/download-url` | `{url}` presigned GET |

| GET | `/floors/:floorId/analysis` | `{status,confidence,version,error?}` |

## Placements (editor)  `/floors/:floorId/placements`

## AI service internal API (`services/ai`, not public)
| POST | `/analyze` | `{imageUrl|imageB64, floorId, units, ruleConfig?}` → `AnalysisResult` |
| POST | `/suggest` | `{rooms[],zones[],ruleConfig?}` → `{placements[]}` (rule engine only) |
| GET | `/health` | model load status |

### `AnalysisResult` (validated by Pydantic ↔ Zod in shared)
```jsonc
{
  "floorId": "…",
  "image": { "width": 2480, "height": 1754 },
  "rooms": [{ "label":"majlis","type":"majlis","polygon":[[0.1,0.2],…],"centroid":[0.2,0.3],"area":0.08,"confidence":0.82 }],
  "zones": [{ "type":"gate","geometry":{"kind":"point","coords":[[0.5,0.98]]},"confidence":0.77 }],
  "detections": [{ "class":"door","bbox":[x,y,w,h],"confidence":0.9 }],
  "placements": [{ "deviceCode":"CCTV","position":{"x":0.05,"y":0.05},"rotation":45,"rationale":"external corner overlooking gate","confidence":0.7,"source":"ai" }],
  "confidence": 0.78,
  "provider": "cv",
  "warnings": ["low DPI page 3"]
}
```

All AI-produced rooms/zones/placements carry `source` and `confidence`; the API persists them as **unreviewed suggestions** the user can accept, edit, or delete. Nothing the AI returns is final.
